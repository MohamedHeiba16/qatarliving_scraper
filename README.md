# qatarliving_scraper

### About 

Spider built by [playwright](https://playwright.dev/), [pandas](https://pandas.pydata.org/docs/) to scrape all data such `title`, `phone`, `price`, `e-mail` form [Qatar_living](https://www.qatarliving.com/)
